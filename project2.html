Links

  • Live Demo

Description

ユーザー同士がお互いに処方された薬品を指定された時間に処方したかを見守れるソフトウェアです。

アメリカ人のWeb開発者4人とチームを組んで開発しました。私はReactを用いたフロントエンド開発を担当し、FDA APIを利用して薬品の情報をレンダーする部分などを担当しました。

Functionalities

  • ログイン機能
  • FDA APIによる薬品情報の表示
  • 友達ユーザーの薬品の服用状態の確認

Tech Stack

  • React
  • Ruby on Rails
  • FDA API
